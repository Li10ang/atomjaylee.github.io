    Aug 4, 2016

    日常使用git一般借助 tortoisegit/sourcetree等图形界面操作git，但为了逼格也需要知道最常用的git命令

    • git init将文件夹变成git仓库

    • add到暂缓区

      • git add readme.md添加单个文件
      • git add .添加所有的文件(不包括空文件夹)
      • git add *.md添加所有的.md文件
